What's the purpose of machine learning?

Practitioners often describe the field as a statistical science, trying to make models to predict correct labels for data points not seen before. The word "model" here refers to any technique (either an algorithm, a set of operations or something else) made for a particular prediction problem. For instance, a model might be something that predicts is there a cat in an image the model has never seen before. Such models are used everywhere, from self-driving cars to cancer screenings.

The more precise task of machine learning practitioners is often to make accurate predictions that will be used by some end user to make decisions. For example, a model that predicts if some person will recommit crime after given probation has been used to sway the opinion of judges in America. Such usecases often get less technical and more ethical.

Unfortunately, there is only so little that humans can accomplish by ingenuity in comparison with a consistent evaluation over millions of choices carried out automatically by a model. This "brute force" approach appears infeasible, but it's actually one of the most important intuitions for machine learning, and more specifically deep learning. Neural networks can often be thought of as nonparametric models, which given enough epochs and data can fit correctly labeled data within an $\epsilon$ error to the ground-truth. The remaining question is: Can it do so for never-before-seen data too?

This is the troubling "Generalization problem". Often, ML engineers rely on intuitions and past references to engineer better models. Then, there is a mismatch between theoretical confirmation and the actual results DL networks are achieveing. In some sense, freely speaking, DNNs are performing better than what the best possible performance math can justify or prove plausible.
